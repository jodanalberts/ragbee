      Welcome to RAGbee

      Build collections of knowledge, then chat with them.

      1. Click + New under Collections
      2. Add files or notes to it
      3. Create a thread and start chatting

      Chunks

      Type and send. RAG is on when a collection is attached.

      Settings

      Your API keys are stored locally in ragbee/data/config.json. Pick chat and embedding providers independently.

      New collection

      A collection is a library of files and notes. You can attach one to a thread later.

      New thread

      A thread is a conversation. Optionally give it a name and pick a collection to ground it in.
